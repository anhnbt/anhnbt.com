## Development

Ứng dụng có 2 file cấu hình môi trường `production` và `development`. Mặc định chạy môi trường `development`

```
./mvn spring-boot:run
```

## Production
Đầu tiên cần build project ra file jar.
```
mvnw clean package
```
hoặc bỏ qua test

```
mvn clean package -DskipTests
```

```
nohup java -jar blog-1.1.6-SNAPSHOT.jar --spring.profiles.active=prod > anhnbt-prod-1.1.6.log &
tail -1000f /opt/anhnbt-prod-1.1.6.log
nohup java -jar blog-1.1.6-SNAPSHOT.jar --spring.profiles.active=staging > anhnbt-staging.log &
```

`Nohup` là một lệnh Unix cho phép chạy ứng dụng ngay cả khi ngắt kết nối.

`&` là **shell's backgrounding operator**
